# DetiGo bot
<b>DetiGo_bot</b> — разработанный на Python (Aiogram 3) Telegram-бот для автоматизации бронирования услуг детской перевозки. Реализован полный цикл взаимодействия с клиентом: от проверки подписки на канал до отправки оформленной заявки администраторам.

- Разработан интуитивный пошаговый диалог с использованием FSM
- Реализована система интерактивных inline-клавиатур для удобного выбора опций
- Интегрирована проверка подписки на канал как условие доступа к боту
- Добавлена валидация вводимых данных (номер телефона, дата)
- Создана систему мультивыбора с динамическим обновлением интерфейса
- Организована модульная структура проекта для простоты поддержки
- Реализована отправка структурированных заявок в административную группу

Проект демонстрирует работу с асинхронным программированием, проектированием UX для Telegram-ботов, обработкой состояний и интеграцией с Telegram API.
